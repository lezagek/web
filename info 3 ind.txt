Задание 1
Создать числовой список длиной n. Заполнить его случайным образом.

Создать макрос для создания группы числовых полей для вывода элементов списка в виде вектора-строки с элементами input. Количество полей в блоке равно n.

Макрос в качестве параметра получает имя и список. Макрос создает input для каждого элемента списка. Имя элемента формируется так: через подчеркивание 
указывается имя и индекс элемента списка. Значение – элемент списка с индексом, указанным в имени поля.

Пример. Пусть список b заполнен следующим образом:

b = [2, 6, 1]

Задание 2
Создать список из n аргументов и значений двух функций:

f(x) = sin(x)y(x) = cos(x)
на интервале [a, b].
Структура списка:

[ { "x": значение,
   "f(x)": значение,
   "y(x)": значение
 },
 { "x": значение,
   "f(x)": значение,
   "y(x)": значение
 },
 ...
]
Создать макрос для формирования таблицы на основе созданного списка.

В качестве параметров макрос должен получать список, номер первой строки для вывода, номер последней строки для вывода (диапазон вывода).

Шапка таблицы должна формироваться из ключей словаря (x, f(x), y(x)), а значения строк – на основе значения ключей.

В таблицу вывести строки, номера которых принадлежат указанному в качестве параметра диапазону. Если заданы несуществующие номера первой и/или последней 
строки, то таблица должна быть выведена полностью.
Проверить работу макроса: вывести на страницу строки таблицы в указанном диапазоне.
